GridTableLight Widget - Demonstration

1. Basic GridTableLight

Alapvető GridTableLight widget minimális konfigurációval. Egyszerű szöveges cellák megjelenítése. 
{
    id: 'basicGridTable',
    type: GridTableLightWidget,
    title: 'Basic Table Example',
    pageSize: 0, // No pagination
    enableExport: false,
    headers: [
        { title: 'Name', width: 150 },
        { title: 'Age', width: 80 },
        { title: 'Department', width: 200 }
    ]
}

2. GridTableLight with Pagination

Lapozással ellátott tábla, 5 sor/oldal beállítással. Automatikus lapozó vezérlők megjelenése. 
{
    id: 'paginatedGridTable',
    type: GridTableLightWidget,
    title: 'Paginated Table',
    pageSize: 5, // 5 rows per page
    enableExport: false,
    headers: [
        { title: 'Product', width: 200 },
        { title: 'Price', width: 100 },
        { title: 'Stock', width: 80 },
        { title: 'Category', width: 150 }
    ]
}

3. GridTableLight with Export

Excel export funkcióval ellátott tábla. Export gomb megjelenik a jobb felső sarokban. 
{
    id: 'exportableGridTable',
    type: GridTableLightWidget,
    title: 'Exportable Table',
    pageSize: 10,
    enableExport: true, // Enable Excel export
    freezeHeaders: true, // Freeze headers on scroll

        { title: 'Invoice ID', width: 120 },
        { title: 'Customer', width: 200 },
        { title: 'Amount', width: 100 },
        { title: 'Date', width: 120 },
        { title: 'Status', width: 100 }
    ]
}

5. Advanced Configuration

Komplett konfigurációs példa az összes elérhető funkcióval. 
{
    id: 'advancedGridTable',
    type: GridTableLightWidget,
    title: 'Advanced GridTable Light',
    skin: 'striped', // 'standard', 'dark', 'striped', 'compact', 'bordered'
    pageSize: 25, // Rows per page (0 = no pagination)
    enableExport: true, // Show export button
    freezeHeaders: true, // Sticky headers
    hideIfNoData: false, // Hide widget if no data
    columnCount: 4, // Override column count
    rowHeight: 40, // Fixed row height in pixels
    borderTop: true, // Show top border
    borderBottom: true, // Show bottom border
    width: '100%', // Widget width
    headers: [
        { title: 'Column 1', width: 150 },
        { title: 'Column 2', width: 200 },
        { title: 'Column 3', width: 120 },
        { title: 'Column 4', width: 180 }
    ]
}

Performance Optimizations

Nagy adathalmazokhoz szánt optimalizálások:
  • Direct Cell Rendering: Nem használ widget példányokat minden cellához
  • Efficient Pagination: Csak az aktuális oldal renderelése
  • Minimal DOM Manipulation: CSS alapú stílusok és transzformációk
  • Memory Management: Proper cleanup on widget destroy
  • Event Delegation: Efficient event handling for large datasets

Repository Integration

MDX lekérdezések és lapozás: A widget automatikusan kezeli az MDX URL kiegészítését $top és $skip paraméterekkel. 
// Repository example:
Repository['myGridTable'] = {
    url: (db) => `/api/v1/ExecuteMDX?mdx=SELECT ...`,
    type: 'GET'
};

// The widget will automatically append pagination parameters:
// /api/v1/ExecuteMDX?mdx=SELECT ...&$skip=25&$top=25
